    median_active_athlete_max_acceleration_ms_2 class-attribute instance-attribute ¶

    median_active_athlete_max_acceleration_ms_2: ClubGraphQLField = ClubGraphQLField(
    "medianActiveAthleteMaxAccelerationMs2"
)

    The median of maximum accelerations (in m/s²) across all active club athletes. Calculates each athlete's top acceleration from their session data, then returns the median. Uses continuous interpolation - if there's an even number of athletes, the result will be the average of the two middle values (e.g., [3.0, 5.0] returns 4.0)

    median_active_athlete_max_deceleration_ms_2 class-attribute instance-attribute ¶

    median_active_athlete_max_deceleration_ms_2: ClubGraphQLField = ClubGraphQLField(
    "medianActiveAthleteMaxDecelerationMs2"
)

    The median of maximum decelerations (in m/s²) across all active club athletes. Calculates each athlete's top deceleration from their session data, then returns the median. Uses continuous interpolation - if there's an even number of athletes, the result will be the average of the two middle values (e.g., [3.0, 5.0] returns 4.0)

    median_active_athlete_top_speed_kph: ClubGraphQLField = (

    The median of maximum speeds (in kph) across all active club athletes. Calculates each athlete's top speed from their session data, then returns the median. Uses continuous interpolation - if there's an even number of athletes, the result will be the average of the two middle values (e.g., [30, 34] returns 32.0)
