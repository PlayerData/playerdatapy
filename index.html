    PlayerData API¶

    GraphQL API for accessing athlete, session, performance, and raw GPS data captured on the PlayerData platform.

    Designed for technology partners, clubs, federations, analytics teams, research integrations.

    Quick links¶

    • API Overview — endpoints, schema, sample queries
    • Authentication — OAuth2 grant types
    • Concepts — Organisation, Club, Session, Athlete
    • Metrics — distance, speed, intensity, raw GPS
    • Python SDK — pip install playerdatapy
    • R wrapper — remotes::install_github("PlayerData/playerdatar")
    • Other Clients — curl, JS, Go, anything that speaks HTTP

    At a glance¶

    query SessionDetails($id: ID!) {
  trainingSession(id: $id) {
    startTime
    sessionParticipations {
      athlete { name }
      athleteMetricSet {
        totalDistanceM
        maxSpeedKph
      }
    }
  }
}

    GraphQL endpoint: https://app.playerdata.co.uk/api/graphql. Authenticate via OAuth2 — see Authentication.
