    AthleteRelativeAccelzoneAttributes¶

    Attributes for assigning relative accelzones to athletes

    Fields¶

    Name Type Description
    athleteIds [ID!]! IDs of the athletes
    autoUpdateRelativeAccelzones Boolean Whether to auto update the relative accelzones
    customMaxAccelerationMs2 Float A custom max acceleration override for the athlete
    relativeAccelzonesLowerBounds AccelzoneLowerBoundsInput Lower bound values of the relative accelzones
